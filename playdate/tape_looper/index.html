Tape Looper

Tape Looper is a mini audio recorder for Playdate that lets you load samples recorded from the built-in microphone or from a higher quality external microphone. You can change the playback speed, reverse direction, and set loop points. Recorded samples and loops can be saved to the device and imported back later, or saved to computer via USB cable.

Controls: B to start and stop a recording. A to start and stop playback. While playing left and right set loop cue points, up jumps playback to the start cue point. During playback the crank changes the playback speed and can reverse the audio.

Code: github.com/orllewin/playdate_tape_looper
