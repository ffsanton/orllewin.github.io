Overview

Modular Play is a casual playground game on Playdate for building little music making patches by connecting modules with cables.

It's meant to be fun and not really for serious use but it can create some beautiful sounding music.

Planned Roadmap

  • v1.0.0 - initial release with a good selection of modules and features
  • v1.0.1+ - inevitable bug-fix releases
  • v2.0.0 - additional modules that didn't make 1.0.0 (a more advanced arpeggiator/sequencer)
  • v3.0.0 - Android keyboard app. Using the same technique as Musik aus Sample add support for a standard touchscreen keyboard on Android (and supply the necessary files for anyone to port to iOS or other platforms)

Performance

The Playdate does incredibly well considering how under powered it is, and for most patches it'll perform fine without extra steps, but on larger patches with dozens of modules and automation you will get some UI slowdown; the bigger the patch the bigger the performance hit. Thankfully the audio fidelity isn't impacted and there are some settings you can access to improve things: from the main runtime menu (tap â’¶ in empty space on the patch) scroll down to System, from this menu category you should first try 'Throttling On', this will reduce the amount of animation updates automated modules have. For an even bigger saving select 'Animation Off' which will replace the animation completely with a static icon, the downside being you don't get visual feedback from the automated sliders and knobs.
