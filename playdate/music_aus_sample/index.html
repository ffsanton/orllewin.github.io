playdate beetroot


A live drum machine instrument for Playdate. Works standalone or with an Android device running the Sample Pads partner app. Android integration makes use of an undocumented Playdate serial API which Panic could disable at any time.

Source:

Playdate: github.com/orllewin/playdate_music_aus_sample

Android: github.com/orllewin/playdate_android_pads
