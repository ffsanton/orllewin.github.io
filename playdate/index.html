Playdate


A port of Coracle to Lua to work with the Playdate SDK:

Github


Coracle offers a quick and easy way to develop drawings/sketches for the Playdate, all that's needed is to learn some basic Lua. Unfortunately Lua has no 1st-class class syntax, object orientation is spoofed by various means and there are several 3rd party solutions if you do some research. Coracle uses SECL for creating ad-hoc custom classes for each drawing. For any in-built objects (currently just Vector) Coracle uses the SimpleLuaClasses example structure
