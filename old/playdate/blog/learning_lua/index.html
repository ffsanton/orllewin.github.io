Coracle for Playdate - Learning Lua


Coracle for Lua has so far been developed without properly learning how Lua works by hacking things together as best I can. Hopefully it's not too terrible but Lua is a strange language and there are some things I really need to get a full understanding of if I plan to work with it for any length of time.

No OO

Lua doesn't have an explicit object orientated class structure like most high-level languages, but you can approximate it with some difficulty. Pretty much everything else in the language behaves with few surprises, but field scope/object references are definitely quirky and it's here where I'll focus my learning - what even is a metatable?

Variable Scope

A quick note on scope: you can define a variable anywhere and it'll be visible everywhere: 

-- visible everywhere:
a = 10

function hello()
    -- also visible everywhere:
    b = 20
end

The local keyword limits field visibility to the current block: 

-- visible everywhere:
a = 10

function hello()
    -- scope limited to hello():
    local b = 20
end
