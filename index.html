Orllewin: current projects

  • Radio was mainly developed for my own use but it has a few users who rely on it too. It's what I use all day everyday for music while working, and with the Android Auto implementation when in the car too. Launch PWA
  • Coracle is a cross-platform sketch playground written in Kotlin, mainly targeting the web and Android
  • Coracle for Playdate is a subset of Coracle in Lua, making porting of Coracle Kotlin drawings easier
  • Orbit Clock is a clock plus particle system for the Playdate
  • Calx is a pseudo-3D game for the Playdate
  • Block Blitz is a Playdate port of the classic BBC Micro game
  • Playdate Fonts are Playdate ports of the classic BBC Micro typefaces
  • Daydream Controller - reverse engineering the old VR controller
  • Noise Timer is a noise generator to help you sleep (free .apk download)
  • Lento is an film simulation camera for Android with support for anamorphic lenses
  • Lut Explorer is a companion app to Lento with brand name film simulation LUTs
  • Small Icon is an Android tool for creating 16x16 icons
  • Seren is a browser for the Gemini Protocol
  • SerenFL is an experimental Flutter rewrite of Seren
